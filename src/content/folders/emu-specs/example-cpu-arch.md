This is an example folder document. You can add markdown files to `src/content/folders/your-folder-name/` and they'll automatically appear on the website.

## Features

- Automatic folder detection from directory structure
- Full markdown support with syntax highlighting
- Table of contents for longer documents
- SEO-friendly with proper metadata

## Getting Started

1. Create a new folder in `src/content/folders/`
2. Add markdown files with proper frontmatter
3. Documents appear on `/folder/your-folder-name/`

You can replace this example with your actual content!
